📦

What is a Data Structure?

A data structure is a way of organizing and storing data in a computer so that it can be accessed and modified efficiently.

⚙️

What is an Algorithm?

A step-by-step procedure or set of rules for solving a specific problem. Algorithms manipulate data in data structures.

🔗

Relationship: Data → DS → Algo

Data is organized using Data Structures. Algorithms operate on these structures. Together they form efficient programs.

Types of Data Structures

📋

Primitive

int, float, char, double — basic types directly supported by hardware/OS.

🔢

Linear (Non-Primitive)

Elements arranged sequentially. Array, Stack, Queue, Linked List.

🌲

Non-Linear

Elements not in sequence. Trees, Graphs — hierarchical/network relationships.

📁

File Structures

Data stored in secondary memory. Sequential, Direct, Index-Sequential.

🎯 Why Study Data Structures?

  • Efficient storage and retrieval of data
  • Foundation for designing efficient algorithms
  • Fundamental to software engineering and system design
  • Enables solving complex problems (compilers, OS, databases, AI)
Analysis

Algorithm Complexity

Measuring how efficient an algorithm is in terms of time and space as input size grows.

⏱

Time Complexity

The amount of time taken by an algorithm as a function of input size n. Counts number of operations.

💾

Space Complexity

The total amount of memory space required (including auxiliary space) as a function of input size.

Asymptotic Notations

O (Big-O) — Upper Bound

Worst-case scenario. f(n) = O(g(n)) means f grows no faster than g.

Example: O(n²) for bubble sort

Ω (Omega) — Lower Bound

Best-case scenario. f(n) = Ω(g(n)) means f grows at least as fast as g.

Example: Ω(n) for linear search

Θ (Theta) — Tight Bound

Average-case. f(n) = Θ(g(n)) when both upper and lower bounds are g.

Example: Θ(n log n) for merge sort

Arrays & Array ADT

Sequential organization of elements of the same type stored in contiguous memory locations.

📌 Key Properties

Fixed size • Same data type • Random access O(1) • Contiguous memory • 0-based or 1-based indexing

Array ADT Operations

  • Access: A[i] — O(1) direct access via base address + offset
  • Insert: Shift elements right, then place — O(n) worst case
  • Delete: Shift elements left after removal — O(n) worst case
  • Search: Linear O(n) or Binary O(log n) if sorted
  • Traverse: Visit each element — O(n)

⚠️ Limitations

Fixed size once declared, insertion/deletion O(n), wasted memory if underutilized.

Array ADT — Pseudocode
// Array ADT Abstract Definition
Array A[MAX_SIZE]
int n = current size

// ACCESS: Return element at index i
FUNCTION Access(A, i):
  IF i < 0 OR i >= n: return ERROR
  return A[i]

// INSERT: Add x at position pos
FUNCTION Insert(A, pos, x):
  IF n == MAX_SIZE: return OVERFLOW
  FOR i = n-1 DOWNTO pos:
    A[i+1] = A[i]     // shift right
  A[pos] = x
  n = n + 1

// DELETE: Remove element at pos
FUNCTION Delete(A, pos):
  IF n == 0: return UNDERFLOW
  FOR i = pos TO n-2:
    A[i] = A[i+1]     // shift left
  n = n - 1
array_adt.cpp
#include <iostream>
using namespace std;

class ArrayADT {
    int* arr;
    int maxSize, n;
public:
    ArrayADT(int size) {
        maxSize = size;
        arr = new int[size];
        n = 0;
    }

    // O(1) — direct access
    int access(int i) {
        if(i < 0 || i >= n) throw runtime_error("Index out of bounds");
        return arr[i];
    }

    // O(n) — shift elements right
    void insert(int pos, int x) {
        if(n == maxSize) { cout << "Overflow!\n"; return; }
        for(int i = n-1; i >= pos; i--)
            arr[i+1] = arr[i];
        arr[pos] = x;
        n++;
    }

    // O(n) — shift elements left
    void remove(int pos) {
        if(n == 0) { cout << "Underflow!\n"; return; }
        for(int i = pos; i < n-1; i++)
            arr[i] = arr[i+1];
        n--;
    }

    void display() {
        for(int i = 0; i < n; i++)
            cout << arr[i] << " ";
        cout << endl;
    }

    ~ArrayADT() { delete[] arr; }
};

int main() {
    ArrayADT a(10);
    a.insert(0, 10); a.insert(1, 20); a.insert(2, 30);
    a.display();    // 10 20 30
    a.remove(1);
    a.display();    // 10 30
    cout << a.access(0) << endl; // 10
}
Unit I

Single Variable Polynomial

Representing and manipulating polynomials like 3x³ + 2x² - 5x + 1 using arrays.

📐 Representation

A polynomial of degree n is stored as array where index = exponent, value = coefficient. p[i] = coefficient of xⁱ

polynomial.cpp
#include <iostream>
using namespace std;

const int MAX = 10;

// Evaluate polynomial using Horner's method: O(n)
double evaluate(int p[], int deg, double x) {
    double result = p[deg];
    for(int i = deg-1; i >= 0; i--)
        result = result * x + p[i];
    return result;
}

// Add two polynomials: O(max(m,n))
void addPoly(int a[], int da, int b[], int db, int c[]) {
    int maxDeg = max(da, db);
    for(int i = 0; i <= maxDeg; i++) {
        int ca = (i <= da) ? a[i] : 0;
        int cb = (i <= db) ? b[i] : 0;
        c[i] = ca + cb;
    }
}

// Multiply two polynomials: O(m*n)
void multiplyPoly(int a[], int da, int b[], int db, int c[]) {
    int dc = da + db;
    for(int i = 0; i <= dc; i++) c[i] = 0;
    for(int i = 0; i <= da; i++)
        for(int j = 0; j <= db; j++)
            c[i+j] += a[i] * b[j];
}

void display(int p[], int deg) {
    for(int i = deg; i >= 0; i--) {
        if(p[i] != 0) {
            cout << p[i];
            if(i > 0) cout << "x^" << i << "+";
        }
    }
    cout << endl;
}

int main() {
    // A = 3x^2 + 2x + 1
    int A[] = {1, 2, 3}; int da = 2;
    // B = x + 4
    int B[] = {4, 1}; int db = 1;
    int C[MAX] = {}, D[MAX] = {};

    addPoly(A, da, B, db, C);      display(C, 2);  // 3x^2+3x+5
    multiplyPoly(A, da, B, db, D); display(D, 3);  // 3x^3+14x^2+9x+4
    cout << evaluate(A, da, 2.0);                 // A(2)=17
}
Unit I

Sparse Matrix

A matrix where most elements are zero. Storing all elements wastes memory — we store only non-zero elements.

Problem

A 1000×1000 matrix with only 500 non-zero elements wastes ~4MB storing zeros. Sparse representation needs only ~6KB.

Representation (Triplet)

Store each non-zero as triplet: (row, col, value). First row stores (rows, cols, total_non_zeros).

sparse_matrix.cpp
#include <iostream>
using namespace std;

struct Triplet { int row, col, val; };

class SparseMatrix {
    Triplet data[100];
    int rows, cols, nnz; // non-zero count
public:
    SparseMatrix(int r, int c) : rows(r), cols(c), nnz(0) {}

    void insert(int r, int c, int v) {
        if(v != 0) data[nnz++] = {r, c, v};
    }

    // Transpose: swap row/col indices
    SparseMatrix transpose() {
        SparseMatrix T(cols, rows);
        for(int i = 0; i < nnz; i++)
            T.insert(data[i].col, data[i].row, data[i].val);
        return T;
    }

    // Add two sparse matrices
    SparseMatrix add(SparseMatrix& B) {
        SparseMatrix C(rows, cols);
        int temp[10][10] = {};
        for(int i = 0; i < nnz; i++)
            temp[data[i].row][data[i].col] += data[i].val;
        for(int i = 0; i < B.nnz; i++)
            temp[B.data[i].row][B.data[i].col] += B.data[i].val;
        for(int i = 0; i < rows; i++)
            for(int j = 0; j < cols; j++)
                if(temp[i][j]) C.insert(i, j, temp[i][j]);
        return C;
    }

    void display() {
        cout << "Row\tCol\tVal\n";
        for(int i = 0; i < nnz; i++)
            cout << data[i].row << "\t" << data[i].col << "\t" << data[i].val << "\n";
    }
};
Searching

Linear & Binary Search

Finding an element's position in a collection — the most fundamental operation in CS.

🔍 Search Visualizer
⚡ Speed:
Enter a target value and choose a search algorithm.

Linear Search

Check each element one by one from left to right until target found or array exhausted.

Time Complexity:
Best: O(1) | Worst/Avg: O(n)

Binary Search

Requires sorted array. Repeatedly halve the search space by comparing with middle element.

Time Complexity:
Best: O(1) | Worst/Avg: O(log n)
Pseudocode
// LINEAR SEARCH
FUNCTION LinearSearch(A[0..n-1], target):
  FOR i = 0 TO n-1:
    IF A[i] == target: RETURN i
  RETURN -1   // not found

// BINARY SEARCH (requires sorted array)
FUNCTION BinarySearch(A[0..n-1], target):
  low = 0, high = n-1
  WHILE low <= high:
    mid = (low + high) / 2
    IF A[mid] == target: RETURN mid
    ELSE IF A[mid] < target: low = mid + 1
    ELSE: high = mid - 1
  RETURN -1   // not found
search.cpp
#include <iostream>
using namespace std;

// Linear Search — O(n)
int linearSearch(int arr[], int n, int target) {
    for(int i = 0; i < n; i++)
        if(arr[i] == target) return i;
    return -1;
}

// Binary Search — O(log n), array must be sorted
int binarySearch(int arr[], int n, int target) {
    int low = 0, high = n - 1;
    while(low <= high) {
        int mid = low + (high - low) / 2; // avoids overflow
        if(arr[mid] == target) return mid;
        else if(arr[mid] < target) low = mid + 1;
        else high = mid - 1;
    }
    return -1;
}

int main() {
    int arr[] = {2, 5, 8, 12, 16, 23, 38, 56, 72, 91};
    int n = 10;
    cout << linearSearch(arr, n, 23) << endl; // 5
    cout << binarySearch(arr, n, 56) << endl; // 7
}

💭 Bubble Sort — Theory

Repeatedly swap adjacent elements that are in wrong order. Largest element "bubbles up" to end each pass. Optimized version stops early if no swaps in a pass.

bubble_sort.cpp
void bubbleSort(int arr[], int n) {
    for(int i = 0; i < n-1; i++) {
        bool swapped = false;
        for(int j = 0; j < n-i-1; j++) {
            if(arr[j] > arr[j+1]) {
                swap(arr[j], arr[j+1]);
                swapped = true;
            }
        }
        if(!swapped) break; // already sorted
    }
}
// Time: O(n²) avg/worst  | Space: O(1)

💭 Insertion Sort — Theory

Like sorting playing cards in hand. Pick one card at a time and insert it in correct position among already-sorted cards. Efficient for nearly-sorted arrays.

insertion_sort.cpp
void insertionSort(int arr[], int n) {
    for(int i = 1; i < n; i++) {
        int key = arr[i];
        int j = i - 1;
        // shift elements greater than key
        while(j >= 0 && arr[j] > key) {
            arr[j+1] = arr[j];
            j--;
        }
        arr[j+1] = key; // place key in correct spot
    }
}
// Time: O(n) best, O(n²) avg/worst | Space: O(1)

💭 Selection Sort — Theory

Find the minimum element in unsorted portion and swap it to the front. Simple but always O(n²) — never better than worst case.

selection_sort.cpp
void selectionSort(int arr[], int n) {
    for(int i = 0; i < n-1; i++) {
        int minIdx = i;
        // find minimum in unsorted portion
        for(int j = i+1; j < n; j++)
            if(arr[j] < arr[minIdx]) minIdx = j;
        swap(arr[i], arr[minIdx]); // place min at front
    }
}
// Time: O(n²) always | Space: O(1)

💭 Quick Sort — Theory

Divide and conquer. Choose a pivot, partition array so elements less than pivot go left, greater go right. Recursively sort partitions. Average O(n log n) but O(n²) worst case.

quick_sort.cpp
int partition(int arr[], int low, int high) {
    int pivot = arr[high]; // last element as pivot
    int i = low - 1;       // smaller element index
    for(int j = low; j < high; j++) {
        if(arr[j] <= pivot) {
            i++;
            swap(arr[i], arr[j]);
        }
    }
    swap(arr[i+1], arr[high]); // place pivot correctly
    return i + 1;
}

void quickSort(int arr[], int low, int high) {
    if(low < high) {
        int pi = partition(arr, low, high);
        quickSort(arr, low, pi - 1);
        quickSort(arr, pi + 1, high);
    }
}
// Time: O(n log n) avg | O(n²) worst | Space: O(log n)
Unit II

Stacks

A LIFO (Last In, First Out) linear data structure. Think of a stack of plates — you can only add/remove from the top.

📌

LIFO Principle

Last element inserted is the first to be removed. Only top of stack is accessible.

⚡

All Ops O(1)

Push, Pop, Peek, isEmpty are all constant time — no shifting needed.

🔧

Applications

Expression evaluation, function call management, undo operations, backtracking.

Stack — Pseudocode
INITIALIZE: Stack S, top = -1, MAX_SIZE = n

FUNCTION Push(S, x):
  IF top == MAX_SIZE-1: ERROR "Stack Overflow"
  top = top + 1
  S[top] = x

FUNCTION Pop(S):
  IF top == -1: ERROR "Stack Underflow"
  x = S[top]
  top = top - 1
  RETURN x

FUNCTION Peek(S):
  IF top == -1: ERROR "Stack Empty"
  RETURN S[top]

FUNCTION isEmpty(S): RETURN top == -1
FUNCTION isFull(S): RETURN top == MAX_SIZE-1
stack.cpp
#include <iostream>
using namespace std;

class Stack {
    int* arr;
    int top, maxSize;
public:
    Stack(int size) : maxSize(size), top(-1) {
        arr = new int[size];
    }

    bool isEmpty() { return top == -1; }
    bool isFull()  { return top == maxSize-1; }

    // O(1)
    void push(int x) {
        if(isFull()) { cout << "Overflow!\n"; return; }
        arr[++top] = x;
    }

    // O(1)
    int pop() {
        if(isEmpty()) { cout << "Underflow!\n"; return -1; }
        return arr[top--];
    }

    // O(1)
    int peek() {
        if(isEmpty()) { cout << "Empty!\n"; return -1; }
        return arr[top];
    }

    ~Stack() { delete[] arr; }
};

int main() {
    Stack s(5);
    s.push(10); s.push(20); s.push(30);
    cout << s.peek() << endl; // 30
    cout << s.pop()  << endl; // 30
    cout << s.peek() << endl; // 20
}

Read 'A' → output: A

decimal_to_binary.cpp + infix_to_postfix.cpp
#include <iostream>
#include <stack>
#include <string>
using namespace std;

// ========= Decimal to Binary =========
string decToBinary(int n) {
    stack<int> s;
    if(n == 0) return "0";
    while(n > 0) {
        s.push(n % 2); // push remainder
        n /= 2;
    }
    string result = "";
    while(!s.empty()) {
        result += to_string(s.top()); s.pop();
    }
    return result;
}

// ========= Infix to Postfix =========
int precedence(char op) {
    if(op == '^') return 3;
    if(op == '*' || op == '/') return 2;
    if(op == '+' || op == '-') return 1;
    return 0;
}

string infixToPostfix(string expr) {
    stack<char> s;
    string result = "";
    for(char c : expr) {
        if(isalnum(c)) result += c; // operand → output
        else if(c == '(') s.push(c);
        else if(c == ')') {
            while(!s.empty() && s.top() != '(')
                { result += s.top(); s.pop(); }
            s.pop(); // remove '('
        } else { // operator
            while(!s.empty() && precedence(s.top()) >= precedence(c))
                { result += s.top(); s.pop(); }
            s.push(c);
        }
    }
    while(!s.empty()) { result += s.top(); s.pop(); }
    return result;
}

int main() {
    cout << decToBinary(13) << endl;             // 1101
    cout << infixToPostfix("A+B*C") << endl;  // ABC*+
    cout << infixToPostfix("(A+B)*C") << endl; // AB+C*
}
Unit II

Queues

A FIFO (First In, First Out) linear data structure. Like a line of people — first person to join is first to be served.

🚶

FIFO Principle

First element inserted is first to be removed. Insertions at rear, deletions from front.

⚡

O(1) Operations

Enqueue (rear) and Dequeue (front) are both constant time operations.

💼

Job Scheduling

OS uses queues for process scheduling, printer jobs, network packet handling.

🎮 Queue Visualizer

Queue — Pseudocode
INITIALIZE: Queue Q[MAX], front = -1, rear = -1

FUNCTION Enqueue(Q, x):
  IF rear == MAX-1: ERROR "Queue Overflow"
  IF front == -1: front = 0
  rear = rear + 1
  Q[rear] = x

FUNCTION Dequeue(Q):
  IF front == -1: ERROR "Queue Underflow"
  x = Q[front]
  IF front == rear: front = rear = -1  // last element
  ELSE: front = front + 1
  RETURN x

FUNCTION Front(Q): RETURN Q[front]
FUNCTION isEmpty(Q): RETURN front == -1
queue.cpp
#include <iostream>
using namespace std;

class Queue {
    int* arr;
    int front, rear, maxSize;
public:
    Queue(int size) : maxSize(size), front(-1), rear(-1) {
        arr = new int[size];
    }

    bool isEmpty() { return front == -1; }
    bool isFull()  { return rear == maxSize - 1; }

    // O(1)
    void enqueue(int x) {
        if(isFull()) { cout << "Queue Full!\n"; return; }
        if(front == -1) front = 0;
        arr[++rear] = x;
    }

    // O(1)
    int dequeue() {
        if(isEmpty()) { cout << "Queue Empty!\n"; return -1; }
        int x = arr[front];
        if(front == rear) front = rear = -1;
        else front++;
        return x;
    }

    int frontVal() { return isEmpty() ? -1 : arr[front]; }

    ~Queue() { delete[] arr; }
};

int main() {
    Queue q(5);
    q.enqueue(10); q.enqueue(20); q.enqueue(30);
    cout << q.dequeue() << endl;    // 10 (FIFO)
    cout << q.frontVal() << endl;  // 20
}
Queue Variants

Queue Variants

Several specialized queue types solve specific problems — circular queues, deques, and priority queues.

🔄

Circular Queue

Rear wraps around to front. Solves the "false full" problem of linear queues. Used for buffering.

↔️

Deque (Double-ended)

Insert and delete from BOTH ends. More flexible than stack or queue.

⭐

Priority Queue

Elements dequeued by priority, not arrival order. Basis for Dijkstra's, Prim's algorithms.

📋

Job Scheduling

OS schedulers use queues. Round Robin uses circular queue. Priority scheduling uses priority queue.

circular_queue.cpp
class CircularQueue {
    int* arr;
    int front, rear, size, maxSize;
public:
    CircularQueue(int n) : maxSize(n), front(0), rear(0), size(0) {
        arr = new int[n];
    }

    bool isEmpty() { return size == 0; }
    bool isFull()  { return size == maxSize; }

    void enqueue(int x) {
        if(isFull()) { cout << "Full!\n"; return; }
        arr[rear] = x;
        rear = (rear + 1) % maxSize; // wrap around!
        size++;
    }

    int dequeue() {
        if(isEmpty()) { cout << "Empty!\n"; return -1; }
        int x = arr[front];
        front = (front + 1) % maxSize; // wrap around!
        size--;
        return x;
    }
};
priority_queue.cpp
#include <queue>
using namespace std;

// STL Priority Queue (max-heap by default)
int main() {
    priority_queue<int> pq;
    pq.push(3); pq.push(1); pq.push(5); pq.push(2);

    while(!pq.empty()) {
        cout << pq.top() << " "; // prints: 5 3 2 1
        pq.pop();
    }

    // Min-heap (for Dijkstra's etc.)
    priority_queue<int, vector<int>, greater<int>> minPQ;
    minPQ.push(3); minPQ.push(1); minPQ.push(5);
    // top() gives minimum: 1, 3, 5
}

// Job Scheduling with Priority Queue
struct Job {
    int id, priority, burst;
    bool operator<(const Job& other) const {
        return priority < other.priority; // max-priority first
    }
};

void jobScheduling(vector<Job>& jobs) {
    priority_queue<Job> pq;
    for(auto& j : jobs) pq.push(j);
    while(!pq.empty()) {
        Job j = pq.top(); pq.pop();
        cout << "Running Job " << j.id << endl;
    }
}
